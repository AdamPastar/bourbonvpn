**📖 Короткий гайд по подключению:**  
1️⃣ Скачиваем конфиг `.yaml` от бота (1 конфиг на 1 устройство)  
2️⃣ Запускаем установленное приложение:  
   - В боковом меню выбираем **Профили**  
   - Нажимаем «+» сверху  
   - В появившемся окне: **Дополнительные настройки → Тип профиля: Локальный**  
   - Загружаем скачанный конфиг  
3️⃣ Переходим в **Настройки → Системный прокси** и включаем  
4️⃣ Возвращаемся на главную страницу и запускаем VPN
