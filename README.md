**Короткий гайд по подключения:**
1) Скачиваем конфиг .yaml от бота (1 конфиг на 1 устройство)
2) Добавляем его в профили в установленном приложении
3) Переходим в настройки и ищем системный прокси, затем включаем его
4) Теперь можно просто запустить VPN и наслаждаться
